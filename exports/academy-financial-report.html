05 · Offline Generation System — Build & Runtime Costs OFFLINE PLAN

The offline generation system consists of four concentric rings. Only the server-side rings (3 & 4) consume OpenAI tokens. Rings 1 & 2 run entirely locally at zero API cost.

1
DETERMINISTIC SEED ENGINE
Seeded PRNG replaces Math.random(). NPC behavior, events, quiz ordering are reproducible without any AI call.
API COST: $0.00 / always offline
2
TEMPLATE ENGINE
576 dialogue contexts, 200 event templates, 250 GED question templates. All client-side, no API.
API COST: $0.00 / always offline
3
CONTENT PACK SERVER
Weekly GPT generation of NPC packs, study variants, RSS events. Distributed to all installs.
API COST: $0.043/week · $2.07/year · flat
4
RSS → WORLD PIPELINE
7 RSS feeds parsed into world events, NPC dialogue topics, contextual GED examples.
API COST: Included in Ring 3 pack generation
