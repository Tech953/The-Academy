"Neo-CRT" Aesthetic (Retro + Modern Fusion)
A new generation won't resonate with pure CRT.
So we design a Neo-CRT visual style:
✔ Smoothened scanlines (optional)
Bright, crisp pixel edges with subtle horizontal glow.
✔ Touch-friendly buttons (even if text-based)
Even older users understand button metaphors.
✔ A mix of ASCII + vector micro-icons
Tiny monochrome icons that recall:
* Fallout Pip-Boy
* early MacOS
* PDA/UIs from the 2000s
* Tamagotchi
✔ Minimal animations (slow, relaxing, analog)
* cursor pulses
* text flicker
* screen bloom when switching windows
You keep the CRT identity, but it becomes clean instead of dated.

🟧 2. Dual-Mode UI: "Legacy Mode" and "Student Mode"
Two themes that modulate usability without breaking immersion.
Legacy Mode (for older users)
* blocky terminal
* manual typing commands
* DOS-like file navigation
* fewer icons, more folders
* strong nostalgia appeal
Student Mode (for newer users)
* clickable windows
* dock/taskbar
* chat bubbles
* modern notifications
* drag-and-drop files
Both modes show the same data, just differently.
This respects cognitive preferences.

🟦 3. Retro-Modern Hybrid Widgets
A few interface components that feel modern yet retro:
✔ "Fuzzy Bubble" Notifications
A notification that appears as:
[ ! ] New Message
> from Advisor
Green CRT glow + rounded modern shape.
✔ Toggle-based settings panel
Toggles slide like old mechanical switches, but feel touchscreen-ready.
✔ Animated cursor "blink states"
Cursor changes shape depending on context:
* arrow
* typing bar
* selection box
* scanning ring
It subtly teaches the user how to interact.

* nostalgic (Millennials → school notebooks)
